## 🚀 READY FOR USE

### Immediate Usage
```bash
# Clone and build
git clone <your-repo>
cd azure-tui
just build

# Or download release
curl -L <release-url>/azure-tui-linux-amd64 -o azure-tui
chmod +x azure-tui

# Start using
./azure-tui
# Press '/' to search
# Try: "type:vm", "location:eastus", "web-*"
```

### CI/CD Usage

# Push to trigger CI
git push origin main

# Create release
git tag v1.0.0
